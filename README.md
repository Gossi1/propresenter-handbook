# ProPresenter Operator Handboek

Een compleet, **self-contained HTML-handboek** voor vrijwilligers die ProPresenter willen leren bedienen tijdens kerkdiensten. Het handboek is opgedeeld in 6 secties, van de basisprincipes tot geavanceerde externe controle.

## 🌐 Live versie

[https://gossi1.github.io/propresenter-handbook/](https://gossi1.github.io/propresenter-handbook/)

## 📖 Inhoud

1. **Intro & Het Lagen-Concept** — Het mentale model van doorzichtige lagen
2. **Schermen & De ProPresenter UI** — Navigatie en schermindeling
3. **WorshipTools Planning** — De bron van de waarheid voor de zondag
4. **Een Setlist Bouwen & Bewerken** — Liederen importeren, splitsen en stylen
5. **Live de Dienst Draaien** — Shortcuts, fouten afhandelen, rust bewaren
6. **Geavanceerd: Externe Controle** — Looks, Macro's, Stream Deck & Companion

## 🗂️ Bestanden

- `index.html` — Het volledige handboek (alles-in-één bestand)
- `Zone/` — Screenshots, afbeeldingen en demo-video's
- `screen.png` / `template.png` — Referentie-afbeeldingen

## 🔤 Talen

- 🇳🇱 **Nederlands** (standaard)
- 🇬🇧 **Engels**
- 🇮🇷 **Farsi**

Schakel tussen talen via de knop rechtsboven in de header.

## 💻 Gebruik

Open `index.html` in elke moderne browser. Er is geen internet of server nodig — het werkt volledig offline.

## 📝 Aanpassen

Het handboek is één enkel HTML-bestand. Bewerk het met elke teksteditor en push wijzigingen naar GitHub om de live site bij te werken.

```bash
git add -A
git commit -m "Jouw wijziging"
git push
```
